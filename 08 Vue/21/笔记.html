使用插件 1. 下载 npm install vue-awesome-swiper -S ***这种下载方式是最新版 npm install vue-awesome-swiper@3.1.3 -S 2. 引入 2.1 全局引入 在 main.js *全局引入*/ import VueAwesomeSwiper from 'vue-awesome-swiper' import 'swiper/dist/css/swiper.css' Vue.use(VueAwesomeSwiper) 2.2 按需引入 [局部引入方式：单个组件引入] /*组件方式引用*/ import 'swiper/dist/css/swiper.css' import { swiper, swiperSlide } from 'vue-awesome-swiper' export default { components: { swiper, swiperSlide } 3. 配置 相关文档
